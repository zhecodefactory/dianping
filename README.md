<h2>✏️项目介绍</h2>
基于大众点评搜索以及推荐业务，在使用SpringBoot + mybatis完成用户登录、注册、商家入驻以及结合前端模板搭建运营后台门店服务管理功能后，借助ElasticSearch逐步迭代，完成高相关性搜索服务，并基于spark mllib2.4.4构建个性化千人千面推荐系统
